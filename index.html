Benvenuti nel sito italiano di tpsit. Qui potrete trovare spiegazioni dei concetti fondamentali riguardante come sono fatte le pagine web.

HTML + CSS + JS

HTML è un linguaggio di markup e serve per formattare ed impaginare i contenuti di una pagina web. I CSS, invece, si occupano di gestirne la visualizzazione (cioè il layout). Javascript, infine, è un linguaggio interpretato (quindi non viene compilato) ed eseguito passo-passo direttamente nel browser (Chrome, Safari, Firefox, Edge...).

HTML

E' un linguaggio di markup, serve a formattare ed impaginare i contenuti di una pagina web. Una pagina web risulta quindi formata da testo semplice (il contenuto vero e proprio della pagina) e da una serie di tag (marcatori HTML), che racchiudono porzioni di testo semplice, applicandovi la formattazione desiderata.
Per realizzare una pagina web occorre semplicemente un editor di testo (Blocco Note, Notepad++..) ed editare il file con estensione ".html". Aprendo lo stesso file con un brower, quest'ultimo lo interpreterà e lo mostrerà a video. Il contenuto vero e proprio verrà visualizzato, mentre i tag verranno solamente interpretati e non visualizzati.

CSS

I fogli di stile permettono di impostare il layout delle pagine web, cioè la modalità con cui i contenuti vengono visualizzati all'interno della finestra del browser. I CSS permettono ad esempio di definire lo stile (colore, dimensione...) del testo e, a partire dalla versione CSS3, anche transizione, disposizione in colonne, animazioni, trasparenze, ombre, angoli arrotondati...

Javascript

E' un linguaggio interpretato, non viene compilato dal compilatore ma viene eseguito passo-passo direttamente nel browser. Non si parla quindi di programmi ma di script (insieme di comandi).
