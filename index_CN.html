HTML + CSS + JS

HTML是一个标记语言，它的作用是通过代码改造和分页网页的内容。CSS呢是用来管理和创作网页的布局. Javascript或则JS是一门编程语言然后他会一步一步的在 browser上进行(Chrome, Safari, Firefox, Edge...).

HTML

HTML是一个标记语言，它的作用是通过代码改造和分页网页的内容。所以一张网页的内容其实是普通的字和tag（HTML的一些标记）组成的。
创作一张网页其实很简单，只需要能写字的软件，比如电脑里面的笔记本啊或则专门写代码的软件像Notepad++，Visual Studio Code。。。，写完之后只要把文件扩展名改成".html"就大功告成了。把刚写完的网页用浏览器打开的话它会在一张网页上显示你所写的一切内容。

CSS

CSS是一些层叠样式表，他的功能就是能够让我们设定网页的布局，就是说网页里的内容是怎么样被显示的。比如CSS能让我们改造字体的风格，颜色，大小。。。然后从CSS3我们还能设定内容的位置，按列排列，动画，边角，透明度，影子，让边角变圆。。。

Javascript

Javascript或则JS是一门编程语言然后他会一步一步的在browser上进行。所以他不是程序而是脚本（命令的集合）。
